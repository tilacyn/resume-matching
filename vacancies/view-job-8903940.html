Job Description:

The IT Security Architecture and Engineering team provides many security capabilities as services to the global AIG enterprise. We are looking for a hands-on cloud security expert to join our growing work force. The person will work across the AIG global cloud services, AIG business CIOs and applications teams, network, computer infrastructure and information security teams to secure AIG's global cloud implementation.

The Principal Cloud Technologist will be responsible for:



  • Global Cloud Security Blueprints: Define "security blueprints" for each approved cloud services to secure cloud architecture, for example, AWS S3 security blueprint. Maintain security blueprints and update when new requirements or new function releases.

  • Detect Security Blueprint Violation: Use cloud Service APIs (e.g., AWS APIs), SaaS solutions (e.g., AWS Guard Duty, RedLock, Netspoke) to detect security blueprint violations (e.g., an unauthorized public exposure of AWS S3 bucket). Work with global cloud team and Business CIO team to remediate security violations

  • Cloud Security Tollgate Process: Act as a design authority on cloud security during the "Permission to Build" and "Permission to Operate" process. Ensure all security blueprints are followed and any violations are remediated. Act as an evangelist to drive security blueprint adoption.

  • Cloud Security Tools Evaluation: Based on AIG's cloud security strategy, identify cloud native solutions for security monitoring. Lead POCs for cloud security solutions.

  • Cloud Security Metrics: Define and execute a set of cloud security metrics for AIG's major cloud platform. The security metrics should be based on top cloud security risks and should provide insights into AIG's global cloud deployment footprint.

  • Cloud Incident Responses and Security Operations: Work closely with AIG's global incident response team to define use cases for cloud infrastructure, for example, what to do when a high risk security violation is detected? Define cloud incident response procedure, severities and SLAs.
