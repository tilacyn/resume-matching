Job Description:

CNT is looking for a Computer Programmer to support their team in Lafayette. This is a very exciting opportunity to join a top-notch team of IT professionals and scientists that conducts relevant and objective research, develops new approaches and technologies, and disseminates scientific information needed to understand, manage, conserve, and restore aquatic and coastal ecosystems and their associated plant and animal communities throughout the nation and the world.



ESSENTIAL DUTIES AND RESPONSIBILITIES

The successful candidate will support the team in the following tasks:

 Database Design & Maintenance

o Interact with databases served by Microsoft, Oracle, and/or open source relational database systems.

o Participate with senior team members on database schema design.

 Software design & architecture

o Participate in writing software specifications/requirements documentation with senior team members.

 Documentation

o Assist in creating training materials/user guides to accompany custom built databases and applications.

o Assist in creating use cases and test plans for written software.

o Document developed project components in the government provided code repository ticket and documentation platform.

 Coding

o Write programming code (including all UI controls) using JAVA and other common programming languages. Proficiency in JAVA is an essential requirement for this position.

o Work on assigned tickets within the Advanced Applications Team ticket/issue system.

o Write SQL database queries using, but not exclusively limited to, common programming languages such as MS/SQL or PL/SQL.

o Include "in-code" comments/documentation for all code blocks prior to repository upload.

o All code shall be delivered free and clear of any malicious code.

o All submitted code will be reviewed monthly by senior programmers. Participate in code reviews of other Advanced Application Team members assessing the accuracy, effectiveness, and quality of coding deliverables.

 Testing

o Publish all code to an in-house testing web server for review by Government.

o Assist in creating use cases and test plans for written software.

o Perform the following types of testing on developed software: unit testing, functional testing, performance testing, and usability testing.

 Communications

o Participate in meetings related to application development.

o Interact with other programmers on the Advanced Applications Team and other agency programmers to ensure transparency of projects.

o Provide technical guidance to other Advanced Application Team members where applicable

Deliverables

By the 10th of each month, a monthly report describing the progress and status of work shall be provided following a certain format that will be provided.

CNT Site Lead and Client representative shall be notified immediately of any issues that requires their input.

SUPERVISORY/MANAGEMENT AUTHORITY

N/A. Will defer to the CNT Program Manager and CNT Site Lead, may coordinate with client representative (Task Order Manager)

Job Requirements:
EDUCATION and/or EXPERIENCE

Bachelors degree in computer science and/or relevant practical experience and training.

KNOWLEDGE, SKILLS AND ABILITIES

The successful candidate must be proficient with JAVA as a software development language, T/SQL or PL/SQL as an enterprise database communication language, and English as an oral and written communication language. Additionally, must be proficient with Eclipse, Spring Tools Suite, and/or Visual Studio as a primary development environment. Must be proficient with using version control technology such as Git. A general understanding of GIS environments and geospatial data containers is required to architect, build and maintain geospatially driven web applications. All work products will be delivered free and clear of any malicious code or grammatical errors and reviewed in a timely manner. Programmers shall participate in meetings related to applications development. All activities related to communications shall be thoroughly documented to ensure continuity of information is maintained. Interaction with other programmers on the team and other client/partner programmers are necessary to ensure transparency. Problems related to software use, or data manipulation and management are remedied immediately. In addition to documenting problems within the required reports, the client will be notified immediately of any problems that require government input. Familiarity with the continuous integration / continuous delivery process and software containers is desired, but not required.
