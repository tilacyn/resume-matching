Job Description:

The Database / Data Science Developer will work within an IT project team that provides services to support research related projects in the Department of Anesthesiology. The primary role will become a subject matter expert on existing databases, to help create databases / data warehouses and reports, and create and maintain data science pipelines and analytics that support the research needs of the Department of Anesthesiology.


These responsibilities include: managing input of various data sources (including Enterprise Data Warehouse, departmental, hospital, and multi-center data sources), development of complex SQL stored procedures and tables/views, creation of ETL / data pipelines, coordination with the needs of research and application development teams, and creation of documentation.



Principal Duties and Responsibilities:
In this role, you will:




  • Code SQL, TSQL/stored procedures, functions, tables/views, and database ETL (Extract, Transform, Load) processes that adhere to internal requirements from scratch

  • Model and implement data warehouse data schemas using best practice database design under guidance from team and clinical leads

  • Learn about existing databases for purposes of extracting data or warehousing needs

  • Identify query improvements, recommend indexes, and additional performance enhancements

  • Create simple reports using Epic EMR querying tools for import and/or export simple SQL reports for use with data catalog web site

  • Manage security and access to different views / aspects of database server

  • Create data pipelines automating data extraction, exploratory data analysis, data pre-processing, and statistical analysis with help of statistician and other team members.

  • Participate in code review, code control system, and code standards development

  • Develop project and system documentation, including model diagrams, end-user training documents, data catalogues

  • Become a subject matter expert on the databases and core applications

  • Communicate project status and prepare project administrative materials, e.g., work/task list, status reports, agendas, memos, minutes, other project files

  • Work with the project manager(s) and team to define product requirements and create high-level architectural specifications, ensuring feasibility, functionality, and integration with existing systems/platforms


Skills/Abilities/Competencies Required:




  • Eligible candidates will have at least 1-2 years of extensive SQL/TSQL development experience and/or data science with Python/R, and can demonstrate their knowledge in a practical skill assessment during the interview process.

  • Candidates will be asked to demonstrate robust knowledge of the technical features and complexities of Microsoft SQL (including SQL Server 2008+, SSIS, and SSRS) as well as fundamentals of data science using either Python or R.

  • Experience with .NET, Java/CFML, Python/R, Hadoop/Spark and/or other sever-side application development technologies would be highly advantageous.

  • Knowledge of statistics and/or statistical software (e.g., R, SPSS, SAS, Mathlab) would be highly advantageous.

  • Basic familiarity with client-side web technologies such as Javascript, HTML, XML, SOAP would be helpful.

  • Experience in healthcare or clinical research fields and/or medical or research terminology would be helpful.

  • Proficiency with MS Excel / Access is expected.

  • Interpersonal, organizational and communication skills are required to communicate with clients and work within a team.
