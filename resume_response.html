SOFTWARE ENGINEER
Summary
To work with an aerospace company that will utilize my engineering knowledge and support aspirations of becoming a project manager to support the United States in aerospace engineering and military technology. Seeking mentorship in project management, networking advice, and business investments.
Experience
11/2015 to Current
Software Engineer NASA - Stinger Ghaffarian Technologies Rapid Prototype Labs at NASA Johnson Space Center － Houston, Texas
  • Provided support in creating electronic procedures and generating flight displays.
  • Wrote the backend of a GUI that allowed simplified structuring and writings of electronic procedures in a certain XML structure for use with the flight display simulator engine.
  • Handled large amounts of dependent files for flight simulator engine using SQL and Git.
  • Developed multiple data manipulating tools that convert electronic procedures from text to XML & HTML for functionality with Lockheed Martin parser & T-38 displays.
  • Overlooked and supported the fabrication of the Ascent/Entry Cockpit Simulator with a team of mechanical engineers and held daily scrum meetings.
  • Updated functionality of previous displays and added new functionality using Angular for the Cockpit Display & Control Interface website.
  • Successfully represented a fully functional web application/online version of the simulator.
  • Supported development of interactive video program that immersed astronauts with display telemetry through the use of cockpit switches which also cued certain videos to play in the simulator (crew module separation, parachute deployment, arming pyros etc.).
06/2015 to 08/2015
Engineering Intern National Oilwell Varco － Conroe, Texas
Software developer NOV Downhole's Technical Software Solutions
  • Supported development of online web-application aimed for use with downhole engineers (C++).
  • Verified web-application's fluid dynamics, heat transfer, and structural mechanics equations in MATLAB.
  • Extensive use of Excel data entry.
  • Created macros to quickly update website database using SQL.
  • Took part in scrum meetings.
  • Gained familiarity with sprints and agile program management.
  • Learned how to communicate effectively in technical manners with programming colleagues and how to communicate with higher levels of management in a more relatable and understanding manner.
